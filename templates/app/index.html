Test task for SET-University

Hello! My name is Roman and that is my implementation of test task. In my project, I implemented the functionality based on the requirements of the test task. Role Based Access Control has been implemented, you can see an example of functionality for an administrator, manager and analyst

User role functionality

Administrator - has the ability to delete and edit all tickets.
Manager - has access to change and delete tickets of the group to which he was associated.
Analyst - has the ability to view tickets of a specific group, but cannot change or delete them.
